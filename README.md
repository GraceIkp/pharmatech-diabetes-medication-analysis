# PharmaTech Diabetes Medication Analysis

This project analyses clinical trial data to evaluate the effectiveness of a diabetes treatment compared to a placebo. The analysis focuses on changes in hemoglobin (HbA1c) levels over time and final treatment outcomes. The goal is to assess treatment efficacy and generate insights into patient response.

---

## Project Overview

The objectives of this project are to:

- Compare treatment and placebo group outcomes
- Analyse hemoglobin (HbA1c) level changes over time
- Evaluate final treatment effectiveness
- Explore relationships in clinical data
- Generate healthcare insights

---

## Dataset

The dataset contains clinical trial data for patients receiving either a treatment or placebo.

### Key Variables

- Treatment Group (treatment vs placebo)
- Initial Hemoglobin (initial_hbA1c)
- Final Hemoglobin (hbA1c)
- Change in Hemoglobin
- Time Period
- Adverse Events
